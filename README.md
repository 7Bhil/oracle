# 🧠 Mirage Oracle Engine

> **Le Cerveau Central.**  
> Quatrième pilier de l'écosystème **MIRAGE**, ce module est le centre de corrélation et de décision. Il analyse les données de tous les autres modules pour orchestrer la défense.

## ✨ Fonctionnalités (Planning)

- **Corrélation (Jour 17)** : Relie les vulnérabilités (`scan`) aux alertes (`sentinelle`).
- **Honeypot Dynamique (Jour 18)** : Ordonne à `ghost` de créer des leurres intelligents.
- **Moteur de Décision (Jour 19)** : Automatise l'isolation ou le clonage selon le risque.
- **Apprentissage (Jour 20)** : Améliore les seuils de détection basé sur l'historique.

## 🚀 Utilisation

```bash
python3 correlator.py
```
